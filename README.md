# Spectral-FLIM
Spectral-FLIM is a Python-based Bayesian tool designed to handle the intricacies of Spectral Fluorescence Lifetime Imaging Microscopy (FLIM), providing advanced capabilities for spectral analysis in fluorescence microscopy.
It is built using CuPy and CUDA for accelerated performance on NVIDIA GPUs.

## Requirements
NVIDIA GPU: Minimum recommendation: RTX 3060 or equivalent.
Software: Download the application archive application.zip

## Usage
Download and Unzip: Download application.zip and unzip it to a convenient location.
Run the Application: use the terminal or just click on app.bin.
Select Data: Within the application interface, select the data files you want to analyze. You can use the provided sample data located in the data folder within the unzipped directory.
